# Intelligent Course‐Aware IDE

**简介**  
这是一个集成课程学习与代码执行的智能 IDE，能让学生在同一环境内切换讲义、笔记与代码，提升学习体验。

![项目演示截图]


**主要功能**  
- 在幻灯片内直接修改并运行代码  
- 内置笔记、标注与书签功能  
- 支持实时协作与讨论  
- AI 学习助理：自动摘要、生成练习题、绘制思维导图  
- 进度追踪仪表盘
